# MoneyTrack 

![Swift](https://img.shields.io/badge/Swift-6.0-orange.svg)
![Platform](https://img.shields.io/badge/Platform-iOS-lightgrey.svg)

**MoneyTrack** is a native iOS application built with **SwiftUI** designed to help users effortlessly track daily expenses and gain insights into their spending habits through monthly category analysis.

The goal of MoneyTrack is simplicity: providing a fluid, native experience to manage personal finances without the clutter of complex accounting software.

##  Key Features

* **Quick Expense Logging and Editting:** Intuitive interface to add/edit/delete transactions in seconds.
* **Monthly Analytics:** Visual breakdown of expenses by category for each month(Food, Transport, Utilities, etc.).
* **History View:** Scroll through past months to compare spending trends.
* **Dark Mode Support:** Fully adaptive UI that looks great in light and dark modes.

---

## Tech Stack

* **Language:** Swift 6.0
* **UI Framework:** SwiftUI
* **Charts:** Swift Charts framework
* **Architecture:** MVVM (Model-View-ViewModel)
* **Minimum Target:** iOS 26.0+

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Mac running the latest macOS
* Xcode 16.0 or later
* Running the simulator on iOS 26+

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/antonpanainte/MoneyTrack.git](https://github.com/antonpanainte/MoneyTrack.git)
    ```
2.  **Open the project**
    Navigate to the project folder and open `MoneyTrack.xcodeproj`.
3.  **Build and Run**
    Select your target simulator (e.g., iPhone 17 Pro) and press `Cmd + R`.
